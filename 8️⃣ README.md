# Following Link Extractor (Scrapy)

A Python Scrapy project to extract links from websites and follow them automatically.  
⚠️ **Important:** This project is for **educational purposes only**. Do not use on websites without permission.

---

## Features
- Extract all links from a webpage
- Follow extracted links recursively
- Safe demo / educational version
- Built using Scrapy framework

---

## Requirements

- Python 3.9+
- Scrapy
